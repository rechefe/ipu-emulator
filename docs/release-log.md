# Release Log

This page lists every pull request that has been merged into `master`, newest
first.  It is regenerated automatically by the
[`release-log` workflow](https://github.com/rechefe/ipu-emulator/actions/workflows/release-log.yml)
on every merge.

| # | Title | Merged at (UTC) |
|---|-------|----------------|

*This page will be populated automatically the next time a PR is merged into `master`.*
*To backfill historical data, run `python docs/update_release_log.py` locally with a valid `GITHUB_TOKEN`.*
